# To run this express web service - 

## Install the dependencies
`npm install`

## Build the app
`npm run build`

## Run the app
`npm run dev`

## When spinning on a running server
`npm run start`

## Set the following env variables:
```
NODE_ENV=
PORT=
CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
OPENAI_API_KEY=
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
```
